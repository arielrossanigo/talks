Title
=====
 
My scientific python toolbox

Abstract
========
When starting a new project, it's fundamental to use the right tools. The aim of this talk is to show some of the main characteristics of the tools that I use in projects related to data and machine learning. 
The main subjects we will cover in the talk are: programming environment, data processing, visualizations and Machine Learning
